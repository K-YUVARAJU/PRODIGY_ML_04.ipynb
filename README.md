📊 Customer Segmentation using K-Means Clustering
🚀 Project Overview

This project focuses on customer segmentation using the K-Means Clustering algorithm. The goal is to group customers based on their Annual Income and Spending Score to help businesses understand customer behavior and improve marketing strategies.

🎯 Objective

To apply Unsupervised Machine Learning (K-Means Clustering) to identify different customer groups and patterns in data.

🛠️ Tools & Technologies Used
Python 🐍
Google Colab ☁️
Pandas 📊
NumPy 🔢
Scikit-Learn 🤖
Matplotlib 📈
📂 Dataset

The dataset contains customer information such as:

Customer ID
Annual Income
Spending Score
⚙️ Workflow
1️⃣ Data Preprocessing
Loaded dataset using Pandas
Selected relevant features (Income & Spending Score)
Normalized/cleaned data
2️⃣ Finding Optimal Clusters
Used Elbow Method
Plotted WCSS vs Number of Clusters
Selected optimal K value
3️⃣ Model Training
Applied K-Means Clustering
Grouped customers into clusters
4️⃣ Visualization
Plotted customer clusters using Matplotlib
Visualized different customer groups
📊 Results
Successfully segmented customers into different groups such as:
High income – low spending
High income – high spending
Low income – low spending
Medium segment customers
